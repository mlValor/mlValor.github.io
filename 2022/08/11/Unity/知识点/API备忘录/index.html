2.Quaternion.LookRotation与Quaternion.FromToRotation

两种方式得到的结果完全不一样。

一 .LookRotation 几何意义是朝向变化（默认会保持在forward轴的旋转为0）–多用与角色朝向
二. FromToRotation 几何意义只是描述向量1 转向向量2的旋转四元素 两者没有任何直接关系。
